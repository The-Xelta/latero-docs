# Overview

Latero는 영수증 OCR 기반 구매 기록 관리 및 워런티 리마인더 서비스입니다.

## 프로젝트 소개

사용자가 영수증 사진을 업로드하면 OCR로 구매 정보를 자동 추출하고 저장한다.

구매일, 상품명, 가격, 판매처를 기록하고 제품별 워런티와 반품 가능 기간을 관리한다.

## 문제 정의

사용자는 구매 후 중요한 정보를 쉽게 잊는다.

- 구매일
- 구매처
- 상품명
- 가격
- 워런티 만료일
- 반품 가능 종료일
- 시리얼번호
- 보증서 및 설명서 위치

이로 인해 다음 문제가 발생한다.

- 보증 기간을 놓친다.
- 반품 가능 기간을 놓친다.
- 구매 내역 확인이 어렵다.
- 제품 관련 문서 관리가 번거롭다.

## 해결 방식

Latero는 영수증 이미지를 기반으로 구매 정보를 자동 정리한다.

- 영수증 OCR 분석
- 구매 정보 자동 저장
- 제품 정보 보강
- 워런티 및 반품 기간 계산
- 만료 전 리마인더 알림
- 보증서, 설명서, 시리얼번호 통합 관리

## 핵심 가치

- 구매 기록 자동화
- 워런티 손실 방지
- 반품 기한 관리
- 제품 정보 통합 보관
- 사용자 시간 절약
