# Product Requirements

## 핵심 기능

- 영수증 OCR 자동 등록
- 구매 정보 자동 저장
- 워런티 자동 계산 및 만료 알림
- 반품 기간 관리
- 시리얼번호 저장
- 보증서/설명서 저장
- 제품 정보 자동 보강
- 리마인더 알림

## 기능 상세

### 영수증 OCR 자동 등록

- 사용자는 영수증 사진을 업로드한다.
- OCR로 구매 정보를 추출한다.
- 원본 이미지를 함께 저장한다.

### 구매 정보 자동 저장

- OCR 결과를 기반으로 구매 기록을 생성한다.
- 사용자는 추출된 정보를 저장 전 수정할 수 있다.

### 워런티 관리

- 구매일 기준으로 워런티 종료일을 계산한다.
- 제품 카테고리별 기본 보증 기간을 적용할 수 있다.
- 사용자가 직접 수정할 수 있다.

### 반품 기간 관리

- 판매처별 기본 반품 기간을 적용할 수 있다.
- 반품 종료일 전 알림을 제공한다.
- 사용자가 직접 수정할 수 있다.

### 시리얼번호 저장

- 제품별 시리얼번호를 저장한다.
- 직접 입력을 기본으로 한다.
- 추후 시리얼번호 OCR을 확장할 수 있다.

### 보증서/설명서 저장

- 제품 관련 파일을 첨부한다.
- 보증서, 설명서, 제품 사진, 기타 문서를 저장한다.

### 제품 정보 자동 보강

- 상품명 기반으로 제품 정보를 보강한다.
- 브랜드, 모델명, 카테고리, 기본 워런티 기간을 연결한다.

### 리마인더 알림

- 워런티 만료 전 알림을 보낸다.
- 반품 종료 전 알림을 보낸다.
- 사용자가 알림 시점을 변경할 수 있다.

## 정책 정의

### OCR 결과

- OCR 결과는 초안 데이터로 사용한다.
- 최종 저장 전 사용자가 확인한다.
- 사용자가 수정한 값이 최종 데이터가 된다.

### 워런티 기간

- 기본값은 제품 카테고리 또는 사용자 입력 기준으로 설정한다.
- 자동 계산이 어려운 경우 수동 입력을 받는다.

### 반품 기간

- 판매처 정책이 있는 경우 자동 적용한다.
- 정책이 없는 경우 기본값 또는 수동 입력을 사용한다.

### 알림 시점

- 기본 알림: 만료 30일 전
- 기본 알림: 만료 7일 전
- 기본 알림: 만료 1일 전
