# Roadmap

## Phase 1

MVP 출시 단계다.

### 목표

- 영수증 기반 구매 기록 저장
- 기본 워런티 관리
- 기본 리마인더 제공

### 주요 기능

- 영수증 업로드
- OCR 처리
- 구매 기록 저장
- 제품 상세 관리
- 워런티/반품 날짜 입력
- 리마인더 알림

## Phase 2

자동화와 사용성 개선 단계다.

### 목표

- 수동 입력 최소화
- 제품 정보 보강
- 알림 경험 개선

### 주요 기능

- 제품 정보 자동 매칭
- 브랜드/모델명 자동 보강
- 판매처별 반품 기간 기본값
- 알림 커스터마이징
- 제품 검색 및 필터

## Phase 3

확장 및 수익화 단계다.

### 목표

- 외부 연동 확대
- 유료 기능 도입
- 제휴 모델 검증

### 주요 기능

- 제조사 워런티 연동
- 이메일 영수증 연동
- 가족/팀 공유
- 프리미엄 구독
- 수리/보험 서비스 제휴
