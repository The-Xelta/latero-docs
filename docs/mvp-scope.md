# MVP Scope

## MVP 범위

MVP는 영수증 기반 구매 기록 저장과 기본 리마인더 제공에 집중한다.

## 포함 기능

- 회원 가입 및 로그인
- 영수증 이미지 업로드
- OCR 기반 구매 정보 추출
- 구매 기록 저장
- 구매 정보 직접 수정
- 제품 목록 조회
- 제품 상세 조회
- 워런티 종료일 입력
- 반품 종료일 입력
- 기본 리마인더 알림
- 보증서 및 설명서 첨부

## 제외 기능

- 자동 제품 매칭 고도화
- 판매처별 반품 정책 자동 연동
- 제조사 워런티 API 연동
- 바코드 스캔
- 이메일 영수증 자동 수집
- 가족/팀 공유
- 중고 거래 연동
- 보험/수리 이력 관리

## 우선순위

### P0

- 영수증 업로드
- OCR 처리
- 구매 정보 저장
- 제품 상세 화면
- 워런티 및 반품 날짜 저장
- 리마인더 생성

### P1

- 첨부 파일 저장
- 시리얼번호 저장
- 제품 검색
- 알림 설정

### P2

- 제품 정보 자동 보강
- 판매처별 정책 적용
- 고급 필터
- 통계 화면
