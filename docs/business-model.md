# Business Model

## 구독 모델

기본 기능은 무료로 제공한다.

고급 관리 기능은 유료 구독으로 제공한다.

### Free

- 영수증 업로드 제한
- 기본 구매 기록 관리
- 기본 리마인더
- 제한된 첨부 파일 용량

### Premium

- 영수증 업로드 무제한
- 첨부 파일 용량 확대
- 고급 리마인더 설정
- 제품 정보 자동 보강
- 이메일 영수증 연동
- 가족 공유

## 제휴 모델

제품 구매 이후 필요한 서비스와 연결한다.

## 제휴 가능 영역

- 수리 서비스
- 보험 상품
- 중고 판매 플랫폼
- 제조사 고객센터
- 전자제품 리셀러
- 보증 연장 서비스

## 수익화 방향

- 월 구독료
- 연간 구독료
- 제휴 전환 수수료
- 보증 연장 상품 수수료
