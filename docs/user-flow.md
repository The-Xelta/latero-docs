# User Flow

## 영수증 업로드 흐름

1. 사용자가 영수증 사진을 업로드한다.
2. 서버가 이미지를 저장한다.
3. OCR 처리를 시작한다.
4. 구매 정보를 추출한다.
5. 사용자가 추출 결과를 확인한다.
6. 필요한 정보를 수정한다.
7. 구매 기록을 저장한다.

## OCR 처리 흐름

1. 영수증 이미지 업로드
2. OCR 엔진 분석
3. 텍스트 추출
4. 주요 필드 파싱
5. 상품 후보 생성
6. 사용자 확인
7. 최종 데이터 저장

## 리마인드 흐름

1. 구매 기록 생성
2. 워런티 또는 반품 종료일 계산
3. 리마인더 생성
4. 알림 예약
5. 알림 발송
6. 사용자가 상세 화면 확인
